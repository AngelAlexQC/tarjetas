# Documentación del Proyecto Financiero

Este directorio contiene toda la documentación del proyecto.

## Estructura

- **ICONS.md** - Documentación sobre los iconos y assets del proyecto
- **QUALITY_GATE_STRATEGY.md** - Estrategia de Quality Gates y métricas de calidad
- **REPOSITORIES.md** - Información sobre los repositorios relacionados
- **implementation/** - Documentación de implementaciones técnicas específicas
  - `ssl-pinning.implementation.md` - Detalles de implementación de SSL Pinning
  - `SSL_PINNING_STATUS.md` - Estado actual de SSL Pinning

## Convenciones

- Archivos de implementación técnica van en `implementation/`
- Archivos de estado temporal (STATUS, RESEARCH, ANALISIS) no se commitean (ver .gitignore)
- Documentación general permanece en la raíz de docs/
